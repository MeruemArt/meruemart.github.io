Requisitos

Se requiere que los siguientes parámetros sean previamente creados en la cuenta de AWS donde se llevará a cabo el despliegue:

  • DevOpsAgentAmiId: ID de AMI para agentes de DevOps, se toma de grupo de variables.

  • DevOpsQuarkusAgentAmiId: ID de AMI para agentes de DevOps especializados en Quarkus, se toma de grupo de variables.

  • MainASGAzureDevopsPat: Token de Acceso Personal de Azure DevOps para el grupo de escalado principal.

  • QuarkusASGAzureDevopsPat: Token de Acceso Personal de Azure DevOps para el grupo de escalado de Quarkus.

Estos parámetros son necesarios para configurar correctamente los grupos de autoescalado y asegurar la comunicación con Azure DevOps.
